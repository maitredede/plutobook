V16 · Billion-laughs XML dependant de la version d'expat

Info   Info / dependance

Statut corrige
Fichier cle meson.build:8-11

Nature

Bonne nouvelle : aucun XML_SetExternalEntityRefHandler ni XML_SetParamEntityParsing → XXE non atteignable. Mais meson.build prefere l'expat systeme ; la protection billion-laughs (expansion d'entites internes) depend donc de la version (protegee par defaut a partir d'expat 2.4.0 ; le fallback bundle 2.7.3 l'est).

Localisation

  • meson.build:8-11 — dependency('expat', fallback: ...) prefere l'expat systeme
  • source/xmlparser.cpp:45-64 — aucun handler d'entite externe (XXE non atteignable)

Risque & impact

Impact : DoS memoire par expansion d'entites si l'hote fournit un expat < 2.4.0.

Reproduction

  1. Rendre poc/lol.xml avec un expat < 2.4.0.

Fichiers d'exemple :

  • poc/lol.xml

Correctif propose

meson.build epingle desormais dependency('expat', version: '≥2.4.0', ...) : un expat systeme trop ancien est rejete par meson, qui retombe alors sur le sous-projet bundle (deja protege). En complement (belt-and-suspenders), XMLParser::parse (source/xmlparser.cpp) fixe desormais explicitement, juste apres XML_ParserCreateNS : XML_SetBillionLaughsAttackProtectionMaximumAmplification(parser, 100.0f) et XML_SetBillionLaughsAttackProtectionActivationThreshold(parser, 8*1024*1024) (memes valeurs que les defauts expat ≥ 2.4.0, fixees explicitement plutot que de compter uniquement sur les defauts de la bibliotheque). Ces deux appels sont proteges par #if defined(XML_MAJOR_VERSION) && (… >= 2.4.0) (verifie apres l'inclusion de expat.h) pour rester compilable si le plancher de version venait a etre abaisse. expat.h ne declare ces deux prototypes que si XML_DTD ou XML_GE est defini par l'includer (correspondant a la maniere dont l'expat lie a ete construit) ; un #define XML_GE 1 local (avant l'inclusion) debloque uniquement ces deux prototypes -- ceci ne change rien au comportement de la bibliotheque deja compilee et n'active aucun parsing de DTD/entite externe/parametre : aucun handler de ce type n'est enregistre dans ce fichier (verifie par grep), XXE reste non atteignable.

Configuration (defaut + configurable)

Version minimale d'expat imposee dans le build (≥2.4.0) + appels explicites de protection amplification (facteur max 100x, seuil d'activation 8 MiB) non exposes comme knob utilisateur (valeurs sures alignees sur les defauts expat, correctif de durcissement plutot que fonctionnalite configurable).
